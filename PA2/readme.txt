5. 檔案壓縮方式: tar zcvf b11032055-p2.tgz b11032055-p2
6. 各檔案說明：
	b11032055-p2/main.cpp		          : 主程式
  b11032055-p2/InputDataParse.hpp 	: InputDataParse
	b11032055-p2/InitialSolution.hpp   : Initialize and make the initial graph
	b11032055-p2/HVGraph.hpp 		        : graph
  b11032055-p2/GenerateOutput.hpp     : output the result
	b11032055-p2/SimulatedAnnealing.hpp 		        : SimulatedAnnealing architecture and controling the update of graph 
  b11032055-p2/Timer.hpp : timer
  b11032055-p2/Block.hpp : storing the feature of input block
	b11032055-p2/Makefile						: Makefile

	b11032055-p2/readme.txt		      : this file
  b11032055-p2/


7. 編譯方式說明：        	
    在 b11032055-p1/ 輸入 : make
	   即可在 b11032055-p1 產生 ./genPlot 的執行檔
	
8. 執行、使用方式說明：
    編譯完成後，在 b11032055-p1/ 下會產生genPlot的執行檔,確認放入inputfile(.*txt)後
	執行的方式為: 
		./genPlot [input file name] [output file name]
			ex: ./genPlot input.txt output.gp

    產生output.gp檔案後，若linux環境下有安裝gnuplot，
    即可使用gnuplot執行output.gp檔案，
    	執行的方式為:
		gnuplot output.gp	//使用gnuplot執行並生成output.png
		eog output.png	//開啟生成的圖片
